## 🚀 Deploy on GitHub Pages

1. Create a new repo on GitHub (e.g. `ta9meemkw`)
2. Upload all files keeping the folder structure
3. Go to **Settings → Pages**
4. Set Source to `main` branch, root `/`
5. Your site will be live at `https://yourusername.github.io/ta9meemkw`

## ✏️ How to Customize

### Change WhatsApp number
In `js/main.js` and `index.html`, replace `96599280845` with your number (include country code, no +).

### Change Instagram
In `index.html`, replace `ta9meem.kw` in the Instagram links.

### Change prices
In `index.html`, find the product cards section and update the "From X KD" text.

### Add your logo image
Replace the SVG star in `.logo-emblem` with an `<img>` tag pointing to your logo file.
